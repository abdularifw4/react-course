01

Apa Itu React?

Dasar-dasar · 5 menit baca

React adalah sebuah library JavaScript yang digunakan untuk membangun antarmuka pengguna (user interface). React dikembangkan oleh Meta (sebelumnya Facebook) dan pertama kali dirilis pada tahun 2013. Sejak itu, React menjadi salah satu library frontend yang paling populer di dunia dengan jutaan developer yang menggunakannya setiap hari.

Keunggulan utama React terletak pada konsep component-based architecture. Dalam React, kamu membangun UI dari komponen-komponen kecil yang bisa dipisah-pisahkan dan digunakan ulang. Setiap komponen mengelola state-nya sendiri dan merender dirinya secara independen. Pendekatan ini membuat kode lebih terorganisir, mudah diuji, dan skalabel.

React menggunakan Virtual DOM — representasi ringan dari DOM sesungguhnya di memori. Ketika ada perubahan data, React menghitung perbedaan (diffing) antara Virtual DOM lama dan baru, lalu hanya memperbarui bagian DOM yang benar-benar berubah. Ini membuat aplikasi React sangat cepat dan efisien dibandingkan manipulasi DOM langsung.

React bersifat declarative, artinya kamu cukup mendeskripsikan tampilan yang diinginkan berdasarkan state, dan React akan menangani cara memperbarui DOM-nya. Ini berbeda dengan pendekatan imperative di mana kamu harus menuliskan langkah-langkah detail untuk memanipulasi DOM secara manual.

💡 Analogi: Bayangkan React seperti sebuah restoran. State adalah pesanan pelanggan, Virtual DOM adalah kertas pesanan yang diperbarui, dan Koki (React) hanya membuat ulang makanan yang berubah — bukan seluruh pesanan dari awal. Efisien, bukan?

02

Instalasi & Environment Setup

Praktik · 10 menit

Sebelum mulai coding dengan React, kamu perlu menyiapkan environment development. Prasyarat utama adalah memiliki Node.js (versi 18 atau lebih baru) terinstal di komputermu. Node.js menyertakan npm (Node Package Manager) yang akan kita gunakan untuk mengelola dependensi proyek.

Cara paling cepat dan direkomendasikan untuk membuat proyek React baru adalah menggunakan Vite. Vite adalah build tool modern yang jauh lebih cepat dari Create React App (yang kini sudah deprecated). Vite menggunakan ES modules secara native, sehingga proses build dan hot reload terasa hampir instan.

Setelah proyek terbentuk, kamu bisa langsung menjalankan development server. Vite akan secara otomatis membuka browser dan menampilkan aplikasi React default. Setiap perubahan yang kamu buat pada kode akan langsung tercermin di browser tanpa perlu refresh manual — ini disebut Hot Module Replacement (HMR).

Terminal - Membuat Proyek React dengan Vite 
# Install Node.js terlebih dahulu (kalau belum ada)
# Lalu buat proyek React baru dengan Vite
npm create vite@latest my-react-app -- --template react

# Masuk ke folder proyek
cd my-react-app

# Install semua dependensi
npm install

# Jalankan development server
npm run dev

# Buka browser di http://localhost:5173
Struktur Folder Proyek React 
my-react-app/
├── node_modules/       # Dependensi proyek
├── public/             # File statis (favicon, dll)
├── src/
│   ├── App.jsx         # Komponen utama
│   ├── App.css         # Styling komponen utama
│   ├── main.jsx        # Entry point aplikasi
│   └── index.css       # Global styles
├── index.html          # Template HTML utama
├── package.json        # Konfigurasi & dependensi
├── vite.config.js      # Konfigurasi Vite
└── README.md

⚠️ Perhatian: Pastikan kamu menggunakan Node.js versi 18 atau lebih baru. Cek versimu dengan perintah node --version. Versi lama mungkin menyebabkan error saat instalasi.

03

Konsep Virtual DOM

Konsep inti · 8 menit baca

Untuk memahami mengapa React begitu cepat, kamu perlu memahami Virtual DOM. DOM (Document Object Model) adalah representasi tree dari halaman web yang digunakan browser. Manipulasi DOM langsung itu mahal — setiap perubahan bisa memicu reflow dan repaint yang membuat halaman menjadi lambat.

React menyelesaikan masalah ini dengan membuat salinan DOM di dalam memori, yang disebut Virtual DOM. Ketika state berubah, React membuat Virtual DOM baru, lalu membandingkannya dengan versi sebelumnya menggunakan algoritma diffing. React kemudian hanya memperbarui node-node DOM yang benar-benar berubah — proses ini disebut reconciliation.

Prosesnya bisa disederhanakan menjadi tiga langkah: (1) State berubah, (2) React membuat Virtual DOM baru, (3) React membandingkan Virtual DOM lama dan baru, lalu menerapkan perubahan minimal ke DOM sesungguhnya. Pendekatan ini sangat efisien karena manipulasi DOM sesungguhnya diminimalkan.

Satu konsep penting lainnya adalah one-way data flow. Data dalam React mengalir dari atas ke bawah — dari komponen parent ke komponen child melalui props. Ini membuat alur data lebih mudah diprediksi dan di-debug dibandingkan two-way binding yang digunakan framework lain.

Ilustrasi: Cara Kerja Virtual DOM 
// Contoh sederhana: tanpa Virtual DOM (cara lama)
// Setiap perubahan langsung memanipulasi DOM
document.getElementById('title').textContent = 'Halo Baru';
document.getElementById('counter').textContent = '1';
// Repaint terjadi 2 kali ❌

// Dengan React Virtual DOM:
// React menghitung diff lalu memperbarui sekaligus
function App() {
  const [title, setTitle] = useState('Halo');
  const [count, setCount] = useState(0);

  // State berubah → Virtual DOM di-update → 
  // React hitung diff → DOM aktual di-update sekali ✅
  return (
    <div>
      <h1>{title}</h1>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>
        Tambah
      </button>
    </div>
  );
}

💡 Analogi: Virtual DOM itu seperti draf foto di Photoshop. Alih-alih mengubah foto asli berkali-kali (yang lambat dan merusak kualitas), kamu buat salinan virtual, edit sebanyak mungkin di layer-layer, lalu flatten sekali saja ke gambar asli. Hasilnya lebih cepat dan lebih bersih!

04

Hello World Pertamamu

Praktik · 7 menit

Sekarang saatnya menulis kode React pertamamu! Kita akan membuat komponen sederhana yang menampilkan "Hello World" dan sedikit interaksi. Buka file src/App.jsx di proyek yang sudah kamu buat, dan ganti isinya.

Komponen React pada dasarnya adalah fungsi JavaScript yang mengembalikan elemen UI (dalam format JSX yang akan kita pelajari di Bab 2). Setiap komponen bisa menerima data dari luar melalui props dan memiliki data internalnya sendiri melalui state.

Dalam contoh di bawah, kita menggunakan useState — sebuah React Hook yang memungkinkan komponen fungsional memiliki state. Hook ini mengembalikan array dengan dua elemen: nilai state saat ini dan fungsi untuk mengubahnya. Ini adalah fondasi dari interaktivitas di React.

src/App.jsx — Hello World dengan Interaksi 
import { useState } from 'react';
import './App.css';

function App() {
  // State untuk menyimpan nama
  const [nama, setNama] = useState('Dunia');
  // State untuk menghitung klik
  const [klik, setKlik] = useState(0);

  const handleClick = () => {
    setKlik(klik + 1);
  };

  const handleNameChange = (e) => {
    setNama(e.target.value);
  };

  return (
    <div style={{ padding: '2rem', textAlign: 'center' }}>
      <h1 style={{ color: '#27d890', fontSize: '2.5rem' }}>
        Halo, {nama}! 🎉
      </h1>
      <p style={{ color: '#9ca3af', marginTop: '1rem' }}>
        Ini adalah aplikasi React pertamamu.
      </p>

      <div style={{ marginTop: '2rem' }}>
        <input
          type="text"
          value={nama}
          onChange={handleNameChange}
          placeholder="Ketik namamu..."
          style={{
            padding: '0.75rem 1rem',
            borderRadius: '0.75rem',
            border: '2px solid #243056',
            background: '#111827',
            color: '#f3f4f6',
            fontSize: '1rem',
            width: '250px'
          }}
        />
      </div>

      <div style={{ marginTop: '1.5rem' }}>
        <button
          onClick={handleClick}
          style={{
            padding: '0.75rem 1.5rem',
            borderRadius: '0.75rem',
            background: '#27d890',
            color: '#0a0f2c',
            fontWeight: 'bold',
            border: 'none',
            cursor: 'pointer',
            fontSize: '1rem'
          }}
        >
          Klik aku! ({klik} kali)
        </button>
      </div>

      {klik >= 10 && (
        <p style={{ color: '#f59e0b', marginTop: '1rem' }}>
          🎯 Kamu sudah klik {klik} kali! Semangat belajar React!
        </p>
      )}
    </div>
  );
}

export default App;

⚠️ Kesalahan Umum: Jangan lupa mengimpor useState dari 'react' sebelum menggunakannya. Error yang muncul biasanya: "useState is not defined". Selalu pastikan baris import ada di atas file.
