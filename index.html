❓ Pertanyaan Umum

Apakah kursus ini gratis?

Ya, 100% gratis! Semua materi tersedia secara open source di GitHub. Anda bisa belajar kapan saja, di mana saja.

Apa prerequisite yang dibutuhkan?

Anda perlu memahami dasar HTML dan CSS. Chapter 1 akan membahas JavaScript dasar yang dibutuhkan React, jadi pemula JavaScript pun bisa mengikuti.

Apakah sertifikat disediakan?

Tidak ada sertifikat resmi, tapi pengetahuan yang Anda dapatkan jauh lebih berharga. Selesaikan semua chapter dan Anda akan siap kerja sebagai React Developer!
